# 灯 · Lantern

> 一盏你为自己点的灯。

Lantern 是一个**个人成长笔记本**——给在迷雾中前行的创业者、独立工作者、自我探索者使用。

它不是又一个笔记软件，也不是又一个生产力工具。它的设计哲学是：**重新认识并再一次养育自己**。

---

## 它是什么

Lantern 由三个相连的舱室组成，每个舱室有一位 AI 伙伴陪着你：

```
┌─ 一 · 洞察 ──────────────────────────────────┐
│                                              │
│   你扔进去：视频、截图、链接、文字、语音      │
│   拾光人帮你：提炼原子想法、自动建立关联       │
│                                              │
│   ↓ 知识不只是被存下，是被看见                 │
└──────────────────────┬───────────────────────┘
                       │
┌─ 二 · 灵感 ──────────┴───────────────────────┐
│                                              │
│   你抛出问题、困惑、好奇                       │
│   点灯人帮你：调取你过去的想法 + 反问 + 给视角  │
│                                              │
│   ↓ 答案不在外面，在你已经知道但还没看清的地方 │
└──────────────────────┬───────────────────────┘
                       │
┌─ 三 · 输出 ──────────┴───────────────────────┐
│                                              │
│   你必须产出：博客、视频、产品、build in public │
│   守灯人帮你：保持节奏、给情绪价值、推你发出去  │
│                                              │
│   ↓ 输出反过来喂养洞察，闭环开始转动           │
└──────────────────────────────────────────────┘
```

---

## 为什么做这个

我在做智能帆板教练的硬件创业。每天遇到大量信息、产生大量想法、陷入大量迷茫。

现有的工具都不够好用：
- 笔记软件让我**记录**，但不帮我**理解**
- AI 助理让我**问答**，但不**了解我**
- 项目管理工具让我**完成任务**，但不**陪我长大**

我需要一个能跟我一起长的容器。它知道我所有的来路，能在我迷茫时把我的过去拿出来给我看；能在我兴奋时帮我冷静；能在我拖延时推我一把。

所以我自己做一个，并把它开源出来。如果你也是在不确定中前行的人，欢迎用它，欢迎改它。

## 设计哲学

**1. 单机优先 · Local-first**
你的数据存在你自己电脑里。Lantern 不需要账号、不上传服务器、不分析你。这是一个只服务你一个人的工具。

**2. 养育，不是管理**
项目管理工具的隐喻是工厂——任务流水线、KPI、效率。Lantern 的隐喻是花园——节奏、季节、耐心。

**3. AI 是伙伴，不是助理**
三个 AI 代理有自己的人格：拾光人克制安静、点灯人苏格拉底式追问、守灯人实干有节奏感。它们不端茶倒水，它们跟你一起想问题。

**4. 输出是义务**
2 周一迭代是硬性原则。70 分版本就要发出去。完美主义是这个工具拒绝服务的东西。

**5. 跨界推送**
洞察舱会偶尔推送跟你兴趣**正交**的内容（你搞硬件就推一篇古典音乐结构）。真正的洞察来自跨界。

## 现在能用什么

**v0.1（已发布）**：纯 HTML 单文件原型，双击浏览器打开即可使用。

- ✅ 灵感卡片捕获（内容、来源、标签、心境）
- ✅ 多维度筛选与搜索
- ✅ "抽一张今日启发" 随机回顾
- ✅ 注意力分布统计
- ✅ 数据持久化在浏览器本地

## 一个建议

如果你打算用这个工具，不要把它当软件。把它当一个**陪你长大的对象**。

灯需要油。你的素材就是它的油。
灯需要点。你的好奇就是火种。
灯需要被看见。所以你必须输出。

愿你也成为一个种灯人。
